JSTG
==

JSTG 是一个用于制作类似《东方Project》的弹幕射击游戏的游戏框架。

本人水平不高，这个框架我写得有许多不好的地方，比方说设计思路可能有问题，我也没搞成 npm 包，我是完全不懂什么 npm 之类的这些东西……我只敢说“能跑就行”。

需要注意的是：本框架基于 pixi.js 。部分素材借用自 Simple 的弹幕引擎模板和《东方Project》系列作品。如果您想要利用此框架制作自己的作品或者进行修改、分发等等，请务必注意不要侵犯我的上游版权方。
